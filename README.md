# Foundations: Landing Page
This is my take on the Landing Page project of The Odin Project. The project aims to help me practice building a stylized landing page simple website using HTML and CSS.

### Project Structure
```bash
. (odin-landing-page)
├── images
│   ├── banana.svg
│   ├── kiwi.svg
│   ├── orange.svg
│   ├── potato.svg
│   └── produce.jpg
├── index.html
├── README.md
└── styles.css
```
